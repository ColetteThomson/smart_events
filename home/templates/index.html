Welcome to the SmartEvents Management System (SEMS)!

SEMS will help you match available people resources to your projects.

SEMS also offers a Project Chat which enables project staff to leave their Ideas or Suggestions on the various projects.

Register for a SEMS account

As a project manager you will need to register, so that you can:

  • create, read, update and delete information on your own projects
  • create, read, update and delete people resources on your own projects

As a people manager you will need to register, so that you can:

  • create, read, update and delete information on all project people resources in SEMS

As a site user you will need to register so that you can:

  • view and leave ideas/suggestions on the project chats

How to use SEMS

Select Projects dropdown from the navigation menu

Click Add Project

Complete the form and submit (only project name and start date are mandatory fields)
